# Cursor Landing Page Clone

This project is a static landing page inspired by the Cursor website.
It is built using only HTML and CSS.

The main goal of this project is to practice frontend layout building using real-world UI sections.

## About the Project

This landing page includes multiple sections such as navigation, hero section, feature blocks, testimonials, changelog, highlights, and footer.

The design follows a dark theme and uses CSS Grid and Flexbox for layout structure.

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

No JavaScript or external frameworks are used.

## How to Run

1. Clone or download this repository
2. Make sure all image and font folders are present
3. Open `index.html` in any modern browser

No installation or setup is required.
